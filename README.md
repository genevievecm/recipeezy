
# Recipeezy

A small single page app built from scratch using:
- the [MealDB](https://www.themealdb.com/api.php) API
- React + React Router
- Styled Components
- Webpack

See the [live demo](https://cranky-hopper-1bbf4b.netlify.app/).

Or if you would like to run this project locally you can clone the repo and run the following commands in the root level:

```
$ npm install
$ npm run start
```
